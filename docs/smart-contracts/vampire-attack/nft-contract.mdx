## ENSO 1155 NFT Contract

Enso extends base [ERC-1155 standard](https://docs.openzeppelin.com/contracts/3.x/api/token/erc1155) integrating [Ownable](https://github.com/OpenZeppelin/openzeppelin-contracts/blob/master/contracts/access/Ownable.sol), `onlyHolder()` modifier, and minting associated to creator.


##### ENSO 1155
- Each protocol has their own NFT associated
- Each NFT tokenID will have same [URI](https://docs.openzeppelin.com/contracts/3.x/api/token/erc1155#IERC1155MetadataURI) linked
- Creators identified
- Holders identified(used for utility of NFT)
- Token supply for each ID


### Open source

Contract source code can be found in [Github repository](https://github.com/EnsoFinance/liquidity-migration/blob/master/contracts/nft/Root1155.sol).
